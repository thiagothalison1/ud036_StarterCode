# Entertainment Center
This is the source code for a Movie Trailer website.

## Installation
Please **clone** the project's repository running the following command on a **terminal**:
```
git clone https://github.com/thiagothalison1/ud036_StarterCode.git
```
**Note**: To properly execute the project, make sure you have the [python 2.7.12](https://www.python.org/downloads/release/python-2712/) installed.

## Usage
1. On a **terminal**, go to the **project's root directory**;
2. **Execute** the following command:
```
python entertainment_center.py
```
3. A web browser will open with the entertainment website.

## Adding Your Own Movies
To add your own movies you can edit the **data/movies.json** file. Each movie information must have the following format:
```
"movie_id": {
    "title": "",
    "description": "",
    "poster_img_url": "",
    "trailer_url": "",
    "imdb_page_url": ""
}
```
